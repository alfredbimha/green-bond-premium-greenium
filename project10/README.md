# Biodiversity Risk Scoring Framework

## Research Question
Can we construct a quantitative biodiversity risk score for industries?

## Methodology
**Language:** Python  
**Methods:** Composite index, PCA, sector analysis

## Data
ENCORE biodiversity dependency data, sector classifications

## Key Findings
Mining, agriculture, and forestry sectors score highest on biodiversity dependency risk.

## How to Run
```bash
pip install -r requirements.txt
python code/project10_*.py
```
